# Lucide icons (ISC), vendored from lucide-static 0.468.0.

Owned by this chrome kit: embedded via `ChromeAssets` and referenced by the
icon enum as `icons/*.svg`.

`panel-left-filled` / `panel-right-filled` are Lucide panel outlines with a
diagonal hatch in the pane (not a solid fill). Title bars typically use
hollow = column closed, hatched = column docked.
